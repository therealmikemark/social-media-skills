# Headshots

Drop creator headshots here, one image per expression. The skill auto-selects the
expression that matches each concept's emotional tone.

Recommended set (filenames are matched by name, case-insensitive):

- `confident.jpg` — eye contact, calm, assertive (default)
- `surprised.jpg` — wide eyes, open mouth, bold reaction
- `pointing.jpg` — pointing at the focal element / payoff
- `happy.jpg` — genuine smile, approachable
- `confused.jpg` — furrowed brow, "wait, what?"

Shooting tips for thumbnail-grade headshots:
- Even, distinctive lighting you can reuse across every video (brand consistency).
- Shoot against a plain background so the face cuts out cleanly.
- Slightly exaggerate each expression — neutral faces underperform on CTR.
- Same framing and distance each time so the face sits consistently in the frame.

If only one photo exists, the skill uses it for every concept and recommends adding
the variants above. Nothing here is committed by default — these are personal assets.
